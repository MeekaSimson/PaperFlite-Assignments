# Paperflite Assignment
This repository contains solutions for the three given problems:

1. Single Number – using HashMap to count frequency
2. Remove K Digits – using Stack and greedy approach
3. Word Pattern – using two HashMaps for mapping

Each problem is placed in a separate folder with its own Main.java file.

## How to Run

Compile and run from the parent folder:

javac RemoveKDigits/Main.java
java RemoveKDigits.Main

javac SingleNumber/Main.java
java SingleNumber.Main

javac WordPattern/Main.java
java WordPattern.Main
